# July 16, 2026 New-State Sweep

Fresh sweep for states not already included in the tracker.

## Bottom Line

This pass found **three states that should be added to the tracker**:

- **Maine**: enacted LD 1126 / Public Law 537, a ghost-gun / serial-number / undetectable-firearm law that expressly defines three-dimensional printers and printed unfinished frames or receivers.
- **North Carolina**: active HB 545 / S 409 proposal banning ghost guns and undetectable firearms; official summaries say undetectable firearm includes manufacture through a 3D printing process.
- **Virginia**: enacted HB 40, a ghost-gun / unserialized-firearm law that defines manufacturing to include additive processes and unfinished frames/receivers to include printing.

This pass also found **two watchlist states**:

- **Texas**: SB 1711 directly targeted 3D-printed firearms and digital firearm schematics, but LegiScan marks it dead in committee, so it is not added to the active tracker.
- **Pennsylvania**: HB 1065 is an active privately made firearm / unserialized parts bill. It includes "printing" in the unfinished-frame-or-receiver definition, but this pass did not find an express 3D-printer-specific restriction comparable to North Carolina, Maine, or Virginia. Keep on watchlist.
